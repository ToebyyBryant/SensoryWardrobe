# Sensory Wardrobe

A mobile-first Flutter app that helps sensory-sensitive individuals and their caregivers make comfortable clothing decisions every day — informed by personal comfort history and real-time weather.

**Team:** Bruce Schulz, Nova Denton-Parry, Zach Kahler, Jeremy Kirkpatrick  
**Course:** CIS248 Advanced App Development — Summer 2026

---

## Features

| # | Feature | DFD Process |
|---|---------|------------|
| 1 | User accounts & multi-profile (caregiver → dependent) | P1.0 |
| 2 | Wardrobe catalog with sensory tags & photos | P2.0 |
| 3 | Live weather via OpenWeatherMap API | P3.0 |
| 4 | Daily outfit logging & post-wear comfort rating (1–5) | P4.0 |
| 5 | Smart outfit suggestions (weather + comfort history) | P5.0 |
| 6 | History view & comfort trend charts | P6.0 |
| 7 | Push notification reminders (morning log, evening rating) | P7.0 |
| 8 | Encrypted cloud backup & restore via Firebase | P8.0 |

```

```

## Getting Started

### Prerequisites
- Flutter SDK ≥ 3.0
- Android Studio or Xcode (for device/emulator)
- A Firebase project
- An OpenWeatherMap API key (free tier works fine)

### Setup

1. **Clone and install dependencies**
   ```bash
   flutter pub get
   ```

2. **Configure Firebase**
   ```bash
   dart pub global activate flutterfire_cli
   flutterfire configure --project=YOUR_FIREBASE_PROJECT_ID
   ```
   This regenerates `lib/firebase_options.dart` and places `google-services.json` / `GoogleService-Info.plist`.

3. **Set your OpenWeatherMap API key**  
   The key is stored in Flutter Secure Storage (never hard-coded). On first launch go to **Settings → Weather API Key** and paste it in.

4. **Run**
   ```bash
   flutter run
   ```

### Code generation (after changing providers or models)
```bash
dart run build_runner build --delete-conflicting-outputs
```

## Database Schema (SQLite — DS1–DS5)

| Table | DFD Store | Description |
|-------|-----------|-------------|
| `user_profiles` | DS1 | Account info, preferences, caregiver links |
| `wardrobe_items` | DS2 | Clothing items, sensory tags, warmth level |
| `outfit_logs` | DS3 | Daily outfit selections + weather link |
| `comfort_ratings` | DS4 | Post-wear scores per outfit log |
| `weather_snapshots` | DS5 | Cached OpenWeatherMap responses |
